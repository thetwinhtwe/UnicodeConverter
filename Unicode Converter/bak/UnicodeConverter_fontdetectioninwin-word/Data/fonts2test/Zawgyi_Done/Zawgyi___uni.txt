ဇော်ဂျီယူနီကုဒ်ဖောင့် သမိုင်း
ဇော်ဂျီယူနီကုဒ်ဖောင့် သမိုင်းမှာ ယခုအထိ ရှင်းလင်းခြင်း မရှိသေးပါ။ ဇော်ဂျီဖောင့် annotation များတွင် ကိုဇော်ဝင်းမြတ် ရေးဆွဲသည်ဟုပါရှိပါသည်။ Okisan အဆိုအရ Alpha Info-Tech Co., Ltd. မှ ကိုဇော်ဝင်းမြတ်အား အခကြေးငွေပေး၍ ရေးဆွဲစေခဲ့သည်လို့ ဆိုပါသည် [1]. ။ ဇော်ဂျီယူနီကုဒ်ဖောင့် အလျှင်စလို ဖြန့်ဝေရခြင်းဟာ အခမဲ့ Unicode Font ထုပ်ဝေရန်တာဝန်ရှိသော အများပိုင် Myanmar NLP မှ အချိန်ဆွဲနေခြင်းကိုတွင်းအားပေးရန် အကြောင်းတစ်ရပ် ဖြစ်ပါတယ် လို့ဆိုပါသည်[2]။ ညီလင်းဆက် အဆိုအရ ဇော်ဂျီဖောင့် ကို အမည်မဖော်လိုတဲ့ ပညာရှင် ငါးဦးက ၂၀၀၅ လောက်မှာ စတင်ခဲ့တယ်လို့ သိရပါသည် [3]။ အချို့ပြောဆိုသူများအရ ဦးငွေထွန်း၏ project မှ ကျောင်းသားတစ်ရောက်မှ ပုညခေါ် ဖောင့်တစ်ခု တင်ပြခဲ့စဉ် ဦးသောင်းစုညိမ်း (CEO, Inforithm-Maze) မှ ထိုဖောင့်ကို planet.com.mm မှာ အသုံးပြုခဲ့သည်။ ဦးငွေထွန်း၏ ခွန့်ပြုချက်မရရှိပဲ သုံးသဖြင့် ရပ်စဲခဲ့သည်။ ထို့နောက် ဦးသောင်းစုညိမ်းသည် လူငါးဦးစုစည်းကာ​ Alpha Mandalay တွင် ဇော်ဂျီဖောင့်ကို ထုပ်လုပ်စေခဲ့သည်ဟု ဆိုပါသည်။[ကိုးကာချက်လိုအပ်] Myanmar IT Professional Zawgyi Group discussion များအရ ထိုငါးဦးအဖွဲ့ဝင် ဦးသောင်းစုညိမ်း နှင့် ကိုရဲမြတ်သူ ပါသည်ဟု မှန်းဆနိုင်သည်မှလွဲ၍ အခြားသူများမှာ အဖော်ထုပ်နိုင်ပေ။[4] 

ဇော်ဂျီဖောင့်တွင် Inforithm-Maze မှ ထုတ်ဝေသော ပုညဖောင့် နောက်ဆက် TrueType font နှင့် Alpha Mandalay မှထုတ်ဝေသော OpenType font ဟူ၍ နှစ်မျိုးရှိသည်။ Alpha Mandalay မှ ထုတ်ဝေသော ဇော်ဂျီ ဖောင့် သည် OpenType ဖြစ်သဖြင် Windows system တွင် ပို၍သင့်တော်ပြီး ClearType နည်းပညာကို ကောင်းစွာ အသုံးချနိုင်သည်။ planet forum မှ ဖြန့်ချီသည့် Inforithm-Maze မှ ထုတ်ဝေသော ဇော်ဂျီဖောင့်သည် Arial font ကို အခြေံ၍ ၂၀၀၅ တွင်နောက်ဆုံးထုတ်ဖြစ်ပြီး Alpha Mandalay မှ ထုတ်ဝေသော ဇော်ဂျီ ဖောင့် သည် Tohoma font ကို အခြေံ၍ ၂၀၀၈ တွင်နောက်ဆုံးထုတ်ဖြစ်သည်။ 

မူလဇော်ဂျီ ဖောင့်တွင် "Powerred by MyaZeDi" ဟု မူပိုင်ခွန့်မှတ်တန်း ပါရှိခဲ့ပါသည်။ ၂ဝဝ၅-၁၂-၁၃ နေ့တွင် မြန်မာ အင်ဖော်တတ်၌ ပြုလုပ်သော သတင်းစာရှင်းလင်းပွဲတွင် Solveware Solution မှ မူပိုင်ခွန့် အလွဲသုံးစားမှုဖြင့် Alpha Computer ကို ပြစ်တင်ခဲ့သည်။ [5] ထို့နောက် ရက်ပိုင်းအတွင်း ဇော်ဂျီဖောင့် တွင် "Powerred by MyaZeDi" မူပိုင်ခွန့်မှတ်တန်း ဖြုပ်ခဲ့သည်။ [6] ၂ဝဝရ တွင် Myanmar Computer Professional Association (MCPA) မှပေးအပ်သော Excellent Award ကို မြန်မာ ဇော်ဂျီ ယူနီကုတ်အဖွဲရရှိသည်။[7] 

ဇော်ဂျီယူနီကုဒ်ဖောင့်ကို ၂ဝဝ၅ တွင် Alpha Info-Tech Co., Ltd. မှ စတင် ထုတ်ဝေခဲ့ပါသည်။​ ၂ဝဝ၆ တွင် PDF ပုံနှိပ်ထုတ်ဝေခြင်း အဆင်ပြေရန် ပြုပြင်ခဲ့ပါသည်။ ၂ဝဝ၈ တွင် အသုံးပြုသူများ တောင်းဆို ချက်အရ --်ဘ နောက်တစ်ခုနှင့် bug fix နှစ်ခု ထည့်သွင်းခဲ့ပါသည်။ 

ယနေ့ထိတိုင် ဇော်ဂျီယူနီကုဒ်ဖောင့် ကို Alpha Info-Tech Co., Ltd. မှ တိုးတက်အောင် ပြုလုပ်လျှက်ရှိပြီး ဖြန့်ချိပါသည်။ ဇော်ဂျီယူနီကုဒ်ဖောင့်သည် အသုံးပြုခွင့် လိုင်စင် မပေးသော်လည်း အစဉ်အမြဲ အခမဲ့ ဖြန့်ချိမည်ဟု ဆိုပါတယ် [8]။ www.zawgyi.net သာလျှင် Official ZawGyi Myanmar Font website ဖြစ်ပါသည် ဟုဆိုပါသည် [9]။ 

[ပြင်ဆင်ရန်​] ဇော်ဂျီယူနီကုဒ်နည်းပညာ နှင့် မြန်မာ့အင်တာနက် အကျိုးသက်ရောက်မှူ
ဇော်ဂျီယူနီကုဒ်နည်းပညာ သည် မြန်မာ အင်တာနက် အသုံးပြုမှူကို အရှိန်အဟုန် မြင့်တင်စေခဲ့ပါသည်။ ဇော်ဂျီယူနီကုဒ်ဖောင့် သည် Platnet Myanmar မှစ၍ လူသုံးများလာ ခဲ့ပါသည်။ နောက်ပိုင်း blogger များမှာ ဇော်ဂျီယူနီကုဒ်ဖောင့် ကိုသာ သုံးစွဲကြပါသည် ။ ယခုအချိန်တွင် မြန်မာသုံးစွဲသူ၏ Operating System တွင် ဇော်ဂျီယူနီကုဒ်ဖောင့် ရှိပြီးဖြစ်သဖြင့် သော်လည်းကောင်း အလွယ်တကူ ဇော်ဂျီယူနီကုဒ်ဖောင့်ကို ရရှိနိုင် ခြင်းတို့ကြောင့် မြန်မာဝတ်ဆိုက်များ မှာ download link ပေးရန် မလိုအပ်ပေ။ 

ဇော်ဂျီယူနီကုဒ်ဖောင့်သည် ​ မြန်မာဝတ်ဆိုက်များ သာမက gtalk များတွင်လဲ အသုံးများလျှက်ရှိပါသည်[10]။ သို့သော် စာစီစာ ရိုက်လုပ်ငန်းများတွင် အသုံးပြုခြင်းနည်းပါသည်။ 

ဇော်ဂျီယူနီကုဒ်ဖောင့်သည် ​ ClearType Technology ကိုကောင်းစွာ အသုံးပြုသဖြင့် အခြား Unicode Font များထက် LCD screen တွင် သိသိသာသာ ချောမွတ်လျှက်ရှိပါသည်။ 

ဇော်ဂျီယူနီကုဒ်ဖောင့်၏ အားနည်းချက်များမှာ စံယူနီကုပ်မဟုပ်သည့်အပြင် codepoint ဖောင်းပွခြင်းနှင့် အစဉ်အတည်ကျမရှိခြင်း ဖြစ်သည်။ ဥပမာ စံယူနီကုပ်တွင် ရရစ်ကို codepoint တစ်ခုတည်း ရှိသော်လည်း ဇော်ဂျီတွင် ရှစ်ခု ရှိလေသည်။ 

ဇော်ဂျီယူနီကုဒ်ဖောင့် အသုံးများရခြင်း အကြောင်းအရင်းတစ်ခုမှာ Myanmar ဘာသာနှင့် သက်ဆိုင်သော Unicode ပြုပြင်ခြင်း အပြီးမသတ် နိုင်သောကြောင့် ဖြစ်ပါသည်။ ၂ဝဝ၈ ခု ဧပြီလ ၄ရက်နေ့ပြဌာန်း သော Unicode 5.1 မှာ မူအားဖြင့် ခိုင်မာခြင်းရှိသော် လည်း ဇော်ဂျီယူနီကုဒ်ဖောင့်သည် ​ လောလောဆယ်ထိ လိုက်နာခြင်းမရှိပါ။ လိုက်နာခဲ့လျှင်လည်း ယခုသုံးနေသော် ဇော်ဂျီစာများဖြင့် ကိုက်ညီမှူ မရရှိနိုင်ပေ။ ဇော်ဂျီယူနီကုဒ်ဖောင့်သည် မည်သည့် Unicode version မျှ လိုက်နာခြင်းမရှိသောကြောင့် သုံးစွဲသူများ အနေဖြင့် နောင်တချိန်တွင် သုံးမရမည်လောဟု သံသယများ ရှိပါသည်။ 

ဤဝီကီပီးဒီးယား တွင် ဇော်ဂျီယူနီကုဒ်ဖောင့်၏ သုံးစွဲခြင်း
Web စာမျက်နှာ များသည် ဇော်ဂျီ ကိုသာအများဆုံး အသုံးပြုနေကြသည်။ ယူနီကုပ် ၅.၁ ဖြင့် စတင်ခဲ့သော နံမည်ကျော် my.wikipedia.org မှာ မသုံးသဖြင့် လူသုံးနည်းပါသည်။ လောလောဆယ် ဇော်ဂျီယူနီကုဒ်ဖောင့် သာလျှင် အသုံးပြုကြမည်ဖြစ်ပြီး အမြင်အားဖြင့် ယခုရှိနေသော ယူနီကုပ် ၅.၁ ဖောင့် များထက် များစွာသာလွန်ပါသည်။ ဇော်ဂျီယူနီကုဒ်ဖောင့်၏ သုံးစွဲခြင်းသည် အကောင်းဆုံးမဟုတ်သော်လည်း အသင့်တော်ဆုံး ဖြစ်ပါသည်။ ဤဝီကီပီးဒီးယား၏ အနာဂါတ် ရှင်သန်ရေးကြိုးပန်းမှူကို ဆွေးနွေးခန်းတွင် တွေ့နိုင်ပါသည်။

    ဥပမာ - လိမေ်မာ်၊ အိနြေ်ဒ


မြန်မာယူနီကုတ်နှင့်ပတ်သက်ပြီး ဒဏ်ဍာရီလိုလို၊ ယုံတမ်းစကားယောင်ယောင် ပုံပြောကောင်းသူများ ရိုက်စားလုပ်နေ ကြချိန်တွင် ဖြစ်ရပ်အမှန်ကို ရှင်းပြပေးသွားသည့် ကိုငွေထွန်းကို ကျေးဇူးများစွာတင်မိသည်။



အိုင်တီစကားဝိုင်းတွင် ယနေ့မှတွေ့ရသည့် နောက်တစ်ယောက်မှာ ဒေါက်တာဇော်မိုးလွင် ဖြစ်သည်။ သူ့ကို SM3 မိုဘိုင်းလ် မြန်မာစာပို့စနစ် ဖြင့် လူသိများသည်။ MRTV-4 တွင်လည်း သူတို့အဖွဲ့ ကြိုးစားအားထုတ်မှုများ တင်ပြသွား ဖူးသည်။ ယနေ့ သူကစကားဝိုင်းသို့ ဧည့်သည်အနေဖြင့် ရောက်လာသော်လည်း ရင်ဖွင့်သွားသည်။ သူတီထွင်ထားသည့် မြန်မာစာပို့စနစ်ကို အဖွဲ့တစ်ဖွဲ့က (Receive Only) ဆိုပြီး ယူသုံးထား၍ဖြစ်သည်။

မြန်မာအိုင်တီလောကသည် ဖူးပွင့်စကာလလောက်တွင်သာ ရှိပါလိမ့်မည်။ အညွန့်ခူးစားချင်သူတွေ၊ ပုံမှားရိုက်ချင် သူတွေ၊ ရိုက်စား လုပ်ချင်သူတွေ၊ ခွင်ထွင်သူတွေ ရှိနေပြီ ဆိုသဖြင့် များစွာအံ့အားသင့်သွားမိသည်။ ပန်းတိမ်မှ မတတ် သေးဘူး ရွှေခိုးသင်ဆိုသလို ဖြစ်နေသည့်အတွက် များစွာစိတ်မကောင်းဖြစ်မိသည်။ မည်သည့်နေရာတွင်မဆို ရိုးသားစွာ ကြိုးစားပြီးကိုယ်ကျိုးရှာလျှင် ကောင်းမည်ထင်သည်။ ကိုယ်ကျင့်တရားဆိုသည်မှာ ကိုယ်ကမကျင့်နိုင်ဘဲ ခေါင်းပေါင်းစ ထာင်ပြနေ အကျိုးရှိမည်မထင်။ ဂုဏ်သိက်ခာဆိုသည်မှာ ငွေဖြင့် ဝယ်ယူ၍မရပါ။ အထင်ကြီးအောင် လုပ်ချင်သူတို့အား လူတိုင်း ထင်မကြီးနိုင်ပါ။

မည်သို့ဆိုစေ အိုင်တီစကားဝိုင်းလေးကိုတော့ စဉ်ပုံမှန်ရှိစေချင်သည်။ အိုင်တီ လောက အကောင်းအဆိုး အကြောင်းအကျိုးတွေကို သိမြင်ခွင့်ရသဖြင့် ဆက်လက် တည်မြဲပါစေဟု ဆုတောင်းလိုက်ပါသည်။Tools for Burmese script

အကောင်အထည်ဖော်တီထွင်နိုင်ကြပါတယ်။ Microsoft, Adobe, IBM စသည့် ကုမ်ပဏီများနှင့် Printer စသည့် Hardware ထုတ်လုပ်သည့် ကုမ်ပဏီများပါထုတ်လုပ်တီထွင်နိုင်ကြပါသည်။ ကန့်သတ်ချက်မရှိပါ။ သို့ရာတွင် လိုက်နာဆောင်ရွက်ရမည့်အချက်များရှိပါသည်။
နည်းနာရှိပါသည်။ တစ်မျိုးတစ်ဘာသာလုပ်၍မရပါ။
မည်သို့အကောင်အထည်ဖော်ဆောင်ရွှက်ရမည်ကို သတ်မှတ်ပြဌာန်းပေးထားပါသည်
(မြန်မာဘာသာ ပညာရှင်များဖြင့်ဆွေးနွေးခြင်းရလာဒ်ဖြစ်ပါသည်)။ မြန်မာဘာသာ font ကိုပြုလုပ်ရန်အတွက် မြန်မာပညာရှင်များကိုမေးမြန်းစရာမလိုအောင် (နောင်တွင်) အချက်အလက်အပြည့်အစုံ သတ်မှတ်ပေးထားပါသည်။
အဆိုပါ ယူနီကုဒ်အဖွဲ့အစည်းမှ မြန်မာနိုင်ငံအတွက် အကွက်ချမှတ်ရာတွင်
တိုင်းရင်းသားဘာသာများ(မွန်၊ရှမ်း၊ကရင် စသည့်) အတွက်ပါထည့်သွင်းပေးထားပါသည်။ သူ့နေရာနှင့်သူ အသုံးပြုရပါမည်။ ထင်သလိုသုံး၊ လိုသလိုသုံး လုပ်၍မရပါ။ Microsoft ကဲ့သို့သော ကုမ်ပဏီများမှ မြန်မာစာကိုထည့်သွှင်း အသုံးပြုတော့မည် ဆိုပါက ယူနီကုဒ်အဖွဲ့အစည်း၏စံကိုသာ အသုံးပြုမည်ဖြစ်ပါသည်။
